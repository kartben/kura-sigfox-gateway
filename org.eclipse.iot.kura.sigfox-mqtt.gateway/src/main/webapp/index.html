Sigfox to MQTT gateway

You need to configure your Sigfox backend to setup this application as a callback for Sigfox communications on your Device Group.

You should set up the HTTP POST callback payload to be as follows:

{
  "device" : "{device}",
  "data" : "{data}",
  "time" : {time},
  "rssi" : {rssi},
  "position" : [{lat}, {lng}]
}
